# greasemonkey plugin

## antennasiteskip
アンテナサイトを回避するプラグイン.  

## removeprfeedly
feedlyでPRがついた広告記事を取り除く.

## pocket_straight
Pocketの登録においてタグを付けずにそのままsaveをクリックする処理を行う.  
クリックいらずでPocketに登録できる.

## CopyText
テキストの選択が不可能なサイトで選択を可能にする  

## Kasi-time
[歌詞タイム](http://www.kasi-time.com "歌詞タイム")で歌詞を選択状態にするボタンを追加し、歌詞をコピペ可能にする
